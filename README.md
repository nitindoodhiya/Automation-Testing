# Automation-Testing

A Cucumber and Appium using gherkin steps framework to test Android Apps
To run the project:
  - import in Intellij
  - add cucumber plugin
  - glue Automation.feature with LoginSteps.java
  - start Appium server
  - start the emulator
  - ensure installing Amazon app
  - Run the project
    - open command prompt
       - mvn clean install
       - mvn test
